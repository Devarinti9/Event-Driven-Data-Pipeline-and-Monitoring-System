# Documentation Index

This `docs/` folder contains the project documentation for **Event-Driven Data Pipeline and Monitoring System**.

## Files
- `01-overview.md` — scope, purpose, and expected behavior
- `02-architecture.md` — application structure and data flow
- `03-setup-and-run.md` — startup guidance and runtime assumptions
- `04-api-reference.md` — endpoint details and expected responses
- `05-monitoring-and-data-model.md` — monitoring fields and entity description
- `06-troubleshooting.md` — common runtime issues and fixes

## Current implementation status
This repository is a **portfolio/demo pipeline service** with FastAPI endpoints, a pipeline run model, a simple external fetch/process flow, and an Airflow scheduling artifact. It is designed as a starter implementation rather than a production data platform.
